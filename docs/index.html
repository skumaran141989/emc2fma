What I do here?

Intent of this space:

Given the phase of software engineering towards AI-reliance that led to new field of prompt engineering, there is misunderstanding of usage of AI - technology should augement humans rather suppress thinking and learning.

Core Philosophy:

Languages are communication tools and DSA defines the efficiency for using the tool. The core of software evolution was hands-on research.

We were able to build Chat-GPT, Claude or an AI via manual coding, thinking with human emotions, solving on-line real-time issues with temperament.

In this website, I will:

  • Showcase products built following software engineering skills and good research
  • Share knowledge based on research via experiments covering DSA, Languages, and System Architecture/Design at an in-depth level
